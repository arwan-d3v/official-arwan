## 1. EXECUTIVE SUMMARY
The **Arwan Personal OS** is a unified SaaS mega-platform designed as a high-density, multi-cognitive environment. It seamlessly integrates professional branding (M1), algorithmic trading (M2), immersive learning (M3), and utilitarian micro-tools (M4) under a single architectural shell.

## 2. CORE MODULE ARCHITECTURE

### Module 1: Obsidian Nexus (CV & Portfolio)
*   **Job-to-be-Done**: Credibility and Differentiation.
*   **Key Features**:
    *   Split-pane real-time CV Builder.
    *   "Bento Grid" VIP Portfolios for high-end professionals.
    *   AI-powered CV Optimizer and Cover Letter Generator.
*   **Monetization**: "Blur-to-Desire" loop for Premium Layouts ($4.5/mo).

### Module 2: Quant Enterprise (Algo-Trading)
*   **Job-to-be-Done**: High-Frequency Operation.
*   **Aesthetic**: "Dark Terminal" — Pure Black (`#000000`) with Cyber Green/Crimson accents.
*   **Key Features**:
    *   Multi-pane high-density trading workspace.
    *   Algorithm Fleet control center with pulsing status indicators.
    *   Live PnL Matrix with trade execution micro-animations.
*   **Monetization**: API Exchange keys and Backtesting Engine locked for Pro/Enterprise tiers.

### Module 3: Tech Academy (Immersive Learning)
*   **Job-to-be-Done**: Focus and Retention.
*   **Aesthetic**: "Library" — Soft dark grays with high-legibility typography.
*   **Key Features**:
    *   Cinematic Theater-Mode video player.
    *   Integrated "AI Code Companion" for contextual learning.
*   **Monetization**: AI Companion and VIP source codes locked behind Pro Tier.

### Module 4: Utility Vault (The Toolkit)
*   **Job-to-be-Done**: Efficiency and Practicality.
*   **Aesthetic**: "Bento-Box" grid of micro-apps.
*   **Key Features**:
    *   Video downloaders (YT/IG/TikTok), MP3 converters, and PDF readers.
    *   Integrated Cloud Storage management.
*   **Monetization**: Direct Google Drive integration locked for Free users.

## 3. GLOBAL APP SHELL (THE OS GLUE)

### 3.1 Unified Identity & Billing
*   **Central Node**: Manages 2FA, User Profiles (Avatar/Settings), and cross-module invoicing.
*   **Single Sign-On (SSO)**: Identity carries over across all M1–M4 verticals.

### 3.2 God Mode Command Center
*   **Owner's Dashboard**: High-level financial tracking (Global MRR in IDR/USD).
*   **System Controls**: Master toggles for module health and maintenance.

### 3.3 Global Command Palette (Ctrl+K)
*   **Teleportation**: Instant context switching between modules and tools.
*   **God Mode Shortcuts**: Exclusive hidden admin actions (MRR checks, user management) visible only to the owner.

### 3.4 Cross-Module Notification System
*   **Live Signals**: Unified toast and dropdown system for real-time events (e.g., Trade Executed, CV Downloaded).

## 4. UI/UX STANDARDS & STATE MACHINE

### 4.1 Visual Brand Identity
*   **Primary Accent**: Indigo Blue (`#2B6FF0`).
*   **Enterprise Accent**: "Obsidian Gold" (`#D4AF37`) for God Mode and Company Tier.
*   **Typography**: `Syne` (Headings/Display) + `Inter` (Body) + `JetBrains Mono` (Labels/Code).

### 4.2 Global State Machine Matrix
| State | Visual Communication |
| :--- | :--- |
| `isLoading` | Pulse Skeleton (Linear, 2s) |
| `isSuccess` | Emerald Glow + Checkmark Animation |
| `isError` | Crimson Border + Shake Interaction |
| `isUpdated` | Blue Border Flash (0.8s) |
| `isPending` | Amber Warning Pulse |
| `inProgress` | Indeterminate Progress Bar |

### 4.3 Signature "Arwan" Footer
*   **Terminal Variant**: Slim, fixed status bar (h-10) for M2.
*   **Dashboard Variant**: Rich, multi-column navigational anchor for M1, M3, M4.
*   **Brand Hook**: "Engineered & Copyrighted by Arwan".
